## 3. 변형 문제 원문

이차함수 `f(x)=ax²+bx+c`가 있다. 중심이 함수 `y=f(x)`의 그래프 위에 있고 반지름의 길이가 `2`인 원 중에서, 다음 조건을 만족시키는 중심이 서로 다른 원의 개수는 `5개`이다.

```text
원을 x축의 방향으로 m만큼, y축의 방향으로 m만큼 평행이동한 원이
x축과 y축에 동시에 접하도록 하는 실수 m의 값이 1개 이상 존재한다.
```

이 5개의 원의 중심의 x좌표를 작은 수부터 크기 순서대로 `x1, x2, x3, x4, x5`라 하자.

```text
x1 = 0,
x2 + x3 + x4 + x5 = 25,
f(x1) < 0
```

`f(5)`의 값을 구하면?

```text
① 7   ② 8   ③ 9   ④ 10   ⑤ 11
```

## 4. 정답

③ 9

## 5. 독립 검산

`g(x)=f(x)-x`라 두면 접선 조건은 가능한 중심 x좌표가 `g(x)=-4,0,4`와 연결된다.

`x1=0`, `f(x1)<0`이므로:

```text
g(0)=f(0)-0=f(0)<0
가능한 값 중 음수는 -4
따라서 g(0)=-4
```

원형과 같은 구조로 `g(x)`는 아래로 열린 포물선이고 꼭짓점은 `y=4`에 접한다.

```text
g(x)=4-a(x-p)^2
```

`g(0)=-4`이므로:

```text
4-ap²=-4
ap²=8
```

따라서 `y=-4`와 만나는 두 x좌표는 대칭성에 의해:

```text
0, 2p
```

`y=0`과 만나는 두 x좌표는:

```text
4-a(x-p)^2=0
(x-p)^2=4/a
```

`ap²=8`이므로 `a=8/p²`.

```text
(x-p)^2 = 4/(8/p²)=p²/2
x = p ± p/√2
```

따라서 다섯 x좌표는:

```text
x1 = 0
x2 = p - p/√2
x3 = p
x4 = p + p/√2
x5 = 2p
```

합 조건:

```text
x2+x3+x4+x5
= (p-p/√2)+p+(p+p/√2)+2p
= 5p
```

변형 A-01에서는:

```text
5p=25
p=5
```

최종값:

```text
f(5)=g(5)+5=4+5=9
```

## 6. 유일성 확인

- `x2+x3+x4+x5=25`에서 `p=5`가 유일하게 결정된다.
- 다섯 x좌표는 `0, 5-5/√2, 5, 5+5/√2, 10`으로 서로 다르다.
- 보기 중 `9`는 하나뿐이다.
- 원형과 같은 핵심 아이디어를 유지한다.

## 7. 난이도 판단

원문과 같은 구조이므로 난이도는 원문과 유사한 상 수준이다.

숫자는 `30→25`, `6→5`, `10→9`로 바뀌었지만 풀이 방식은 동일하다.
