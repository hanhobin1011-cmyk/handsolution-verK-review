  - 도형/보조선을 얹는 순서: 원 함수 그래프 → 경계 `a`/꼭짓점 → 수평 보조선 → `h(1)`~`h(5)` 교점 개수 라벨.
  - 강조할 부분과 약하게 둘 부분: 함수 그래프는 파랑/초록으로 선명하게, h 보조선은 연한 회색 점선, h 라벨은 오른쪽 외부 여백에 정렬.
  - 학생 시선 이동 순서: 사인 조각 주기 확인 → `a=18/5` 확인 → 포물선 최솟값 확인 → 수평선별 교점 개수 확인.
