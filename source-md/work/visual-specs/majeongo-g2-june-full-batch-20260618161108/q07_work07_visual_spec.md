- 원본 참고 이미지 경로: `none`

- 학생이 이 그림에서 읽어야 할 관계:
  1. 원 함수 그래프는 `0≤x<2π`의 `4tan(x/4)`와 `2π<x≤4π`의 `2tan(x/4)` 두 가지가 먼저 보인다.
  2. 그 위에 `A-B-C` 직선을 얹고, `B`가 `x=2π` 오른쪽에 있음을 보인다.
  3. 마지막으로 `△AOB`, `△BCP`를 옅게 표시해 넓이비 `7:3` 계산으로 이어진다.
- 교재형 시각 설계:
  - 원 함수 그래프 중 반드시 보존할 부분: 두 탄젠트 조각, `x=2π` 점근선, `O`, `P`, `A`, `B`, `C`의 상대 위치.
  - 도형/보조선을 얹는 순서: 원 함수 그래프 → 점근선/축 → A/B/C/P/O 점 → 직선 AC → 두 삼각형 옅은 음영.
  - 강조할 부분과 약하게 둘 부분: 함수 그래프는 선명한 파랑/초록, 삼각형 음영은 매우 연하게, 직선 AC는 검정 얇은 선.
  - 학생 시선 이동 순서: 구간별 함수 확인 → A와 C의 대칭 조건 확인 → B 위치 확인 → 두 삼각형 넓이비 확인.

## 2. 좌표계/구성 데이터

- 좌표계 정의: x축 `0`부터 `4π`, y축은 교재형 설명을 위해 압축 스케일을 사용하되 A/B/C의 상하·좌우 관계를 보존한다.
- 축 범위/스케일: `x=[0,4π]`, `y`는 A와 C, 원 함수의 주요 조각이 잘리지 않게 표시한다.
- 필수 점: `O`, `P(4π,0)`, `A(a,4u)`, `B(28π/13,0)`, `C(4π-a,-2u)`, `a=20π/13`.
- 필수 선/곡선/호: `x=2π` 점근선, 왼쪽 원 함수 `4tan(x/4)`, 오른쪽 원 함수 `2tan(x/4)`, 직선 `A-B-C`.
- 필수 음영/색칠 경계: `△AOB`, `△BCP`를 매우 연한 색으로 표시하되 원 함수와 축을 가리지 않는다.
- 필수 라벨: `O`, `P`, `A`, `B`, `C`, `x=2π`, `b=28π/13`.
- 라벨 표기 규칙:
  - `π`는 수식 파이로 표기한다.
  - 분수는 가능한 세로분수 형태로 표기한다.
  - 함수값/개수 라벨은 `h(1)`처럼 교육과정 표기로 쓴다.
  - 수식/분수/그래프 라벨은 `scripts.harness.labels`의 kyunghoonmath 렌더링 유틸을 사용한다.
- 금지 라벨/표현:
  - `pi`, `2pi`, `4pi`, `28pi/13`, `h1`, `h2`, `18/5` 단순 가로식, 진한 불투명 음영
