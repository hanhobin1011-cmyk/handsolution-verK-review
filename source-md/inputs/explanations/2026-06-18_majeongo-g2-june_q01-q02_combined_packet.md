## 문제 재작성용 원문

### 1번

`π/2 ≤ x ≤ 3π/2`일 때, 방정식 `2sin x + √3 = 0`을 만족시키는 `x`의 값은?

`① 2π/3  ② 5π/6  ③ π  ④ 7π/6  ⑤ 4π/3`

### 2번

`0 ≤ x < 2π`일 때, 부등식 `2cos x - √2 ≤ 0`을 만족시키는 모든 `x`의 값의 범위는 `α ≤ x ≤ β`이다. `β-α`의 값은?

`① π  ② 7π/6  ③ 4π/3  ④ 3π/2  ⑤ 5π/3`
